Concierge Widget — Embed Test

This page tests the full embed flow:

  1. Loader reads data-widget-id
  2. POSTs to /api/widget/token with this page's origin
  3. Backend validates origin against DB allowlist
  4. JWT issued → iframe injected → chat UI appears

To test the allowed-origin path: add http://localhost:5500 (or wherever you serve this file) to your widget's allowed_origins via the admin page.

To test the blocked-origin path: remove this origin from allowed_origins. The widget should fail silently — this page must not break.
